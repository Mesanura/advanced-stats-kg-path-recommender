## [1.1.0] - 2026-07-15

### Added

- 抽取教师、管理员共用的知识管理组件，新增知识点组合筛选及“知识点 / 前置关系 / 关系图”三个视图。
- 前置关系支持完整新增、编辑和删除；新增事务性 `PUT /api/v1/knowledge/prerequisites/{knowledge_point_id}/{prerequisite_id}` 接口，路径参数标识待替换的旧边。
- 管理员增加“知识图谱”和“推荐策略”治理入口。
- 学生画像与教师诊断详情显示中文“建议学习方向”，并提供明确空状态。
- 学习路径节点增加 `stage`，路径响应增加 `stage_count`，学生端和教师端按阶段分组展示。
- 新增 Alembic `0002` 迁移，旧路径升级后自动标记为 `stale`。
- Playwright 增加桌面和移动双视口、关系完整 CRUD、管理员策略修改、多阶段路径、画布像素、横向溢出和控制台错误验收。

### Changed

- 多个前置关系统一按全部必修处理；推荐集合覆盖目标的全部未掌握祖先，并在完整图传递闭包上进行稳定拓扑排序。
- 未知掌握度按 `0.2` 处理，已掌握祖先跳过；目标已掌握时返回单节点复习路径。
- 超长依赖使用尽量均衡且不超过上限的多阶段计划，路径节点保持全局唯一和全局序号。
- 路径评分继续使用薄弱覆盖、前置衔接、阶段数惩罚和难度跳跃四项配置。
- `suggested_directions` 收紧为能力维度枚举，API JSON 值保持兼容。
- 前后端应用版本统一更新为 `1.1.0`，覆盖率门槛提高到后端 85%、前端语句/行 90%、分支 85%、函数 70%。

### Fixed

- 修复知识点改名冲突返回数据库错误的问题，现统一返回 HTTP 409。
- 登录页路由探测改用匿名安全会话接口，消除预期 401 控制台错误，同时保留 `/auth/me` 的原有兼容行为。
- 修复移动端诊断抽屉、关系对话框和多阶段路径的视口溢出及节点横向挤压。

### Quality

- 后端 49 项测试通过，覆盖率 87.35%，无资源警告。
- 前端 23 项测试通过，语句/行 98.10%、分支 89.79%、函数 77.77%，生产构建通过。
- Playwright 在 `1440x900` 与 `390x844` 下共 6 项通过，覆盖三角色真实流程、画布像素、横向溢出和控制台错误。
- 最终性能实测：规则法 123.518ms、BKT 118.129ms、仪表盘 P95 10.820ms、路径生成 5.748ms。

## [1.0.3] - 2026-07-15

### Fixed

- 修复教师端学生诊断 Drawer 尺寸解析为零，导致只有遮罩而不显示详情面板的问题。
- 学生详情增加显式点击、加载状态和接口错误提示。
- 教师端 E2E 增加诊断抽屉可见性与非零宽度断言。

## [1.0.2] - 2026-07-15

### Fixed

- 统一 PowerShell 与 Bash 的 Python、虚拟环境和 pnpm 自动发现行为。
- PowerShell 安装脚本改为离线优先校验依赖，仅在缺包时执行有界联网安装。
- PowerShell 开发模式在任一服务退出后清理完整进程树，避免 Uvicorn 重载子进程占用端口。
- PowerShell 脚本统一使用 UTF-8 BOM，兼容 Windows PowerShell 5.1 的中文解析。

## [1.0.1] - 2026-07-15

### Added

- 新增 Bash 环境下的安装、开发、测试和单地址运行脚本。
- 支持 Linux、macOS 以及 Windows Git Bash，并自动识别对应虚拟环境与 pnpm 命令。
- 补充 PowerShell/Bash 双环境运行与故障排查说明。

### Fixed

- 修复从 PowerShell 直接调用 Git Bash 时标准工具目录未进入 `PATH` 的问题。
- 优化 Bash 安装脚本的幂等依赖校验，仅在缺少 Python 包时执行有界联网安装。
